# 《蚝田》

- **类型**：现实题材
- **故事调性**：克制抒情、历史厚重、时代变迁

## 主要人物
- **老淡（主角）**：约60岁，蚝农，前生物老师，32年前玩具厂大火中受伤失去生育能力，暗恋"她"32年却从未表白，用蚝田守护城市的记忆
- **男孩**：7岁，"她"的孙子，每周末来蚝田玩，承受教育压力，单纯善良
- **"她"（男孩的奶奶）**：约60岁，32年前玩具厂大火幸存者，老淡的初恋，一直住在海湾边守着老淡
- **男孩的妈妈**："她"的女儿，单亲母亲，对儿子教育焦虑，准备再婚生子
- **侄子（集团公司总经理）**：负责蚝田拆迁补偿，理解老淡的坚守

## 核心梗概
六十岁的蚝农老淡守护葵涌海湾最后一片蚝田三十二年，在蚝田被征用的最后一天，七岁男孩来告别，老淡才意识到他守护的不只是蚝田。

## 导语
安静的葵涌海湾被游艇俱乐部、珊瑚度假村、海洋拓展训练基地包围着，只剩下西南岬角那片金黄色浅滩——东经114°24′，北纬22°38′，太平洋海岸最后一片蚝田。六十岁的蚝农老淡在这里守了三十二年，今天是蚝田的最后一天。七岁的男孩像往常一样来玩，却是来告别的。而老淡不知道的是，守护他三十二年的，其实是那个他从未敢靠近的女人。

## 起（第1章）

**开场情境**：蚝田被征用的最后一天，周末，天气晴朗。60岁的蚝农老淡天没亮就来了，做着那些本不必做的事——检测水质、清理蚝田、加固绳索，都是习惯。

**人物出场**：7岁男孩像往常一样来蚝田玩。男孩说"我长大后会像您一样，种很多蚝"，但老淡知道这不可能——男孩的妈妈给他报了8个兴趣班（国际象棋、绘画、跆拳道、机器人编程、读谱训练、篮球、网球、烘焙），要求"每科都得优"，男孩快要哭出来。

**冲突建立**：
- 老淡面临蚝田消失（征用的最后一天）
- 男孩面临教育压力（8个兴趣班、每科都得优）
- 暗藏的告别气氛（男孩说"我长大后会像您一样"）

**安慰与教导**：老淡用海洋生物的例子开解男孩——旗鱼游得快、燕儿鱼能飞、鱼在深海、格陵兰鲨长寿，"它们都很优秀，但做不到像对方那样，什么都优秀。"他们一起观察蚝进食，男孩情绪缓和。

**章末钩子**：男孩突然说"妈妈想生个妹妹"，透露出新的家庭变动。

**本章要点**：
- 开场冲突：蚝田的最后一天 + 男孩的教育压力
- 人物困境：老淡32年的守护即将结束 + 男孩承受"每科都得优"的焦虑
- 关系呈现：老淡像爷爷一样疼爱男孩，用专业知识和温柔开导他

## 承（第2-3章）

### 第2章

**蚝田的价值**：老淡的蚝田出产顶级生蚝，曾让法国Gillardeau品牌蚝家族的人惊艳。他有稳定顾客——市里两家顶级酒店。

**身份转变**：老淡本名淡仔，32年前是葵涌小学生物老师，"发生了一件事"后离开学校，开始养蚝。那些年他年轻力壮，驾船去海里捞野生幼蚝，南海台风一次次摧毁蚝田，他从不放弃。

**关系揭示**：
- 男孩家三代（奶奶、妈妈、男孩）住在海湾北边老旧小区
- 男孩的爸爸死于2015年光明区工业园塌方事故
- 老淡很少收男孩家的钱，白送了好多年蚝
- 老淡会特别照顾有"十几岁安静女孩"的年轻人

**百日宴往事**：5年前男孩出生，老淡精心准备了100只最好的蚝（48小时净化），但男孩家没有邀请他。老淡推测"女儿肯定和妈妈狠狠吵了一架"，他深感愧疚，怨自己连累了"女人"。

**悬念加深**：老淡与男孩家（特别是奶奶）有隐秘的往事，但他主动回避，甚至故意让对方找不到他。

### 第3章

**作文困境**：男孩说老师要求"写伟大的事情"，他不知道什么叫伟大。老淡讲时代故事——那个年代人们挨饿，后来门窗打开，人们跑到"潮汐丰沛的地方"，"你奶奶就是那个时代的人"。

**压力升级**：男孩透露更多信息：
- 妈妈报了更多兴趣班，他不能再来海边玩
- 脸色苍白得"像椰子浆咖啡蛋糕"
- "我不喜欢妹妹，我会杀了她，就像杀蚝一样"（压力下的极端表达）

**蚝田被征用的背景**：两年前，街道集团公司总经理（老淡的侄子）来谈拆迁。政府开发项目征用了这片蚝田，补偿丰厚，给了一年时间。侄子毕恭毕敬："阿叔，城市会越来越美好，您也该享享福了。"

**蚝桩的秘密**：侄子一字不差地复述：
- 102根蚝桩 = 1993年清水河仓库爆炸15人 + 玩具厂大火87人
- 73座蚝排 = 2015年光明区滑坡77人
- 外海4根蚝绳 = 4个失踪者
- "来了的，走了的，死了的，都得记住，就像海记住所有的鱼虾"

**本章要点**：
- 冲突升级：男孩不只是压力大，还要面对二胎；蚝田被征用的事正式揭开
- 关键信息：蚝桩不是普通养殖工具，而是对城市记忆的守护
- 悬念设置：1993年玩具厂大火与老淡有什么关系？

## 转（第4章）

**32年前的大火**：老淡回忆那天下午，他还是生物老师，在课堂上看到窗外玩具厂方向升起火焰和浓烟。"她"在三楼成衣组，十几岁的女工，他们刚约会一次。他冲向玩具厂救她。

**绝望的救援**：
- 大火封住通道，他攀上三楼却被防盗钢条隔在窗外
- 透过焊死的栅栏，他看见十几岁的小姑娘躺在地上或裹挟火焰
- 她们组织过文学社，请他当老师
- 他拼命撕扯铁丝网（手上肉掉一大块）、踹钢条，机器爆炸将他从三楼掀下

**幸存与失去**：
- "她"幸运地被派去一楼领材料，第一批疏散，她那组只有她活下来
- 老淡坠落后手术摘掉某个严重损伤的器官，失去生育能力
- **因为这个，他主动放弃了"她"**

**现在的呼应**：
- 两年前老淡在外海遇到少年布氏鲸，脱衣下水游向它（最后一次亲近海洋）
- 两个月后布氏鲸死亡
- 侄子来谈拆迁时，老淡已经收了外海漂浮筏，不再下海
- 理由：他不会再"脱光衣裳下到海里去"（接受衰老和告别）

**本章要点**：
- 转折事件：32年前的真相——大火、救援、失去生育能力、放弃恋人
- 证据揭示：蚝桩是对那87个死去女工（包括"她"的组员）的纪念
- 人物抉择：老淡当年选择放弃"她"（因为不能生育），现在选择接受告别（收回漂浮筏）

## 合（第5章）

**最后的收获**：落潮时分，老淡收最后一批货，海鸟盘旋。男孩再次说起妈妈要生妹妹的事，还说"我会杀了她，就像杀蚝一样"。老淡想到自己"杀了多少蚝"，反而感到庆幸——他终于放弃了这片蚝田。

**安慰与遗憾**：老淡安慰男孩"你只是害怕，不习惯"。他没告诉孩子，有些事他也做不好——他不能像蚝那样"分泌黏液依附其上，一生永不脱离"，也不能像它们那样"雌雄同体，为自己产下幼蚝"。

**告别的尝试**：
- 男孩第一次说"我以后不能再来玩了"
- 老淡误解："没有谁能把海搬走"
- 男孩走了，又跑回来纠正："我们要搬家了，搬到市里去"
- 妈妈要给他找"会笑的老师"，"我们把奶奶也带走，她不用守着这儿了"

**真相大白**：老淡顿悟——"她"（男孩奶奶）知道蚝田没了，他会离开，所以她也不必待在这儿了。**"他以为守了'她'32年，其实是'她'守了他32年。"**

**释然与遗忘**：老淡感到一阵轻松。他想，孩子很快会忘记葵涌海湾的一切，"大伙儿可不都这样"。

**结尾照应**：落潮了。海水快乐地说：哗——，我走啦，哗——，我走啦。（呼应开头的"哗——，我来啦。哗——，我走啦"，只剩下离开）

**本章要点**：
- 结局走向：男孩搬家 + 奶奶（"她"）离开 + 蚝田消失
- 主题升华："他以为守了她32年，其实是她守了他32年"
- 代价呈现：32年的守护换来告别，童年记忆将被遗忘
- 余味留存：潮水声、遗忘的预言、生命继续（"一阵轻松"）
