# 人物小传

## 主要角色

**老淡（主角）** 约60岁，被太阳晒成黝黑色，夏天穿牛津布防水裤和亮晶晶的聚酯纤维防晒服，活似某星球生命遗落在海湾的玩具。性格隐忍、执着、温柔，说话简短有力，对海洋生物和孩子都充满耐心。他是蚝农，曾是小学生物老师，精通生物知识，能准确说出每一种海洋生物的特性。32年前在玩具厂大火中从三楼坠落受伤，失去生育能力后主动放弃了初恋"她"，转而守护葵涌海湾最后一片蚝田。他用102根蚝桩纪念1993年两场事故死去的102人，用73座蚝排纪念2015年滑坡事故的77人。为保护这片水域的水质，他去过各种部门上百次，和官员吵架，不吃不喝坐在门口，直到警车和救护车来。他养的蚝曾让法国Gillardeau品牌蚝家族的人惊为天人。在蚝田被征用的最后一天，通过男孩的告别，他才意识到：他以为守了"她"32年，其实是"她"守了他32年。

## 重要配角

**男孩** 7岁，脸色鲜活红晕，瘦小，文静。喜欢蹲在蚝排上和海鸟大声说话，和老淡聊天。在学校学习有点吃力，很少得小红花，但会吹黑管，吹些伤感的曲子。性格单纯善良，对世界充满好奇，但也承受着巨大的教育压力。妈妈给他报了国际象棋、绘画、跆拳道、机器人编程、读谱训练、篮球、网球、烘焙等八个兴趣班，要求"每科都得优"，让他精神涣散，几乎要哭出来。他不想要妹妹，甚至说"我会杀了她，就像杀蚝一样"，但只是因为害怕和不习惯。他是"她"的孙子，也是老淡与"她"之间无法言说的联系。在最后一天，他来蚝田告别，告诉老淡他们要搬到市里去，奶奶也要一起走。

**"她"（男孩的奶奶）** 约60岁，湖北人，16岁来葵涌海边打工。1993年在玩具厂成衣组工作，大火发生前被派去一楼领材料，因此幸存，她那个文学社的组员只有她活下来。她是老淡的初恋，但老淡因失去生育能力而主动放弃她。她后来结婚生女，女儿嫁人后丈夫在2015年光明区工业园滑坡事故中遇难。她一直住在海湾边的老旧小区，在社区捡垃圾，守着老淡32年。直到蚝田被征用，她才决定和女儿、孙子一起搬到市里。她从未在小说中直接出场，只通过男孩的只言片语和老淡的回忆存在。

**男孩的妈妈** 约35岁，"她"的女儿，单亲母亲。丈夫在2015年光明区恒泰裕工业园塌方事故中遇难。她对儿子教育焦虑严重，给孩子报了八个兴趣班，要求"每科都得优"。她准备再婚，想生个女儿。她不知道老淡和母亲的往事，或者知道但选择不提。五年前儿子百日宴时，她"肯定和妈妈狠狠吵了一架，而且大哭了一场"，因为没有请老淡。她决定搬到市里，找一个"会笑的老师"给孩子，带着母亲一起离开海湾。

**侄子（街道集团公司总经理）** 老淡五服内的侄子，负责蚝田拆迁补偿工作。他理解老淡的坚守，能一字不差地复述蚝桩和蚝排的意义："102根蚝桩，15根是1993年8月5日清水河仓库爆炸死去的15个人，87根是同年11月19日咱们镇大火烧死的87个玩具厂女工……您说过，来了的，走了的，死了的，都得记住，就像海记住所有的鱼虾，不然对不起人家。"他吞吞吐吐告诉老淡拆迁的事，毕恭毕敬说"阿叔，城市会越来越美好，您也该享享福了"，承诺公司会选个好地方安置那些蚝桩。他代表新一代的实用主义和对老一辈的尊重。

## 其他角色

**少年布氏鲸** 两年前出现在外海，用尾巴搅起海水捕食。老淡脱去外套下到水里，向它游过去。两个月后死亡。它象征着老淡与海洋、与自然、与生命力的最后联系。

**那些死去的女工** 1993年玩具厂大火中死去的87个女工，十几岁的小姑娘，她们组织了文学社，请老淡当老师。诗人某梅16岁、诗人某云霞17岁、剧作家某小芳21岁……她们的名字模糊但存在，通过102根蚝桩被记住。
