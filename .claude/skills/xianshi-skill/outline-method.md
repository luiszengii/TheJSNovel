# 现实题材大纲创作方法论

## 总体取向

现实题材小说重"**真实可感**"与"**社会总体性**"，以贴近当下的生活细节与人物命运为核心。

**核心原则**：
- 聚焦时代议题（职场困境、城乡差异、公共事件、代际关系等）
- 追求"可共鸣"：让读者看到自己或身边人的影子
- 兼顾爽点与现实触感，但**不脱离生活逻辑**
- 以关键节点反转推动情节，**拒绝悬空设定和不可能事件**

---

## 整体创作公式
```
现实题材短篇 = 可验证的困境 + 具体的压力源 + 不可兼得的选择 + 有代价的结局
```

---

## 一、核心梗概写作

### 公式
```
核心梗概 = 人设标签 + 现实困境 + 核心冲突
```

### 写作要点
| 要素 | 要求 | 正确示例 | 错误示例 |
|------|------|---------|---------|
| **人设标签** | 职业+年龄+特征 | "38岁的中层HR" | "一个普通上班族" |
| **现实困境** | 可验证的压力 | "面临裁员和房贷断供" | "遇到了人生低谷" |
| **核心冲突** | 两难选择 | "在职场规则和良心之间" | "面临选择" |

### 优秀示例
- "38岁的HR在裁员名单上看到恩师的名字，签字则保住职位，不签则丢掉饭碗"
- "离婚协议书上的房产分割条款，让独生女不得不在父母之间做出选择"
- "拆迁款到账那天，村会计发现账目和实际对不上，查还是不查"

## 二、导语写作

### 公式
```
导语 = 爆点事件 + 具体细节（数字/地点/职业术语）+ 悬念钩子
```

### 写作要点
| 要素 | 作用 | 写法 |
|------|------|------|
| **第一句** | 开门见山抛冲突 | 直接写事件，不铺垫 |
| **中间部分** | 增强真实感 | 用数字、地名、职业术语 |
| **最后一句** | 制造悬念 | 留下关键问题或倒计时 |

### 时代议题的融入

导语要触及读者关切的社会话题：

| 议题类型 | 具体方向 |
|---------|---------|
| **职场困境** | 35岁+危机、裁员潮、加班文化、职场PUA、劳动纠纷 |
| **城市生活** | 房贷压力、教育焦虑、医疗困境、养老问题、消费陷阱 |
| **代际关系** | 二胎家庭、独生子女养老、婆媳矛盾、原生家庭创伤 |
| **公共事件** | 拆迁补偿、社区治理、环境维权、邻里纠纷 |

**目标**：让读者产生"这事可能发生在我身边"的代入感（可共鸣）

### 示例对比
| 类型 | 内容 |
|------|------|
| ✅ **正确** | "体检报告上的阴影，让李医生想起三年前那场医疗纠纷。2024年6月，杭州某三甲医院，他的恩师因为同样的误诊赔了50万。而现在，他手里的这份报告，属于那位恩师的女儿。他必须在今晚8点前做出决定。" |
| ❌ **错误** | "命运的齿轮开始转动，他面临着艰难的抉择。这是一个关于救赎的故事..." |

## 三、起承转合结构设计

### 【起】第1章：诱因与困境（2000-3000字）

#### 写作重点
```
起 = 快速入局 + 建立3层压力 + 困境外化 + 选择点
```

| 模块 | 写作要点 | 具体方法 |
|------|---------|---------|
| **快速入局** | 开场即冲突 | 第一段就出现：裁员通知/体检报告/法院传票/催款单 |
| **建立压力源** | 3个可量化压力 | 压力1：金钱（房贷月供12000）<br>压力2：责任（老人住院）<br>压力3：时间（合同7天后到期） |
| **困境外化** | 用动作呈现焦虑 | 反复看手机/撕纸巾/摁笔<br>环境细节：催缴单/未接来电/堆积文件 |
| **章末钩子** | 具体事件逼出选择 | 电话响了/门开了/邮件来了 |

#### 禁忌
- ❌ 大段内心独白
- ❌ 铺垫日常生活
- ❌ 上帝视角评论

---

### 【承】第2-3章：推进与加压（每章2000-3000字）

#### 第2章写作重点
```
第2章 = 主动出击 + 小成果 + 新问题 + 压力升级
```

| 模块 | 写作要点 | 具体方法 |
|------|---------|---------|
| **主动出击** | 符合职业的行动 | HR查劳动合同/医生翻病历/会计对账 |
| **小成果+新问题** | 表面解决，深层矛盾 | 找到漏洞→发现漏洞后面更大的坑<br>拿到证据→证据会牵连无辜 |
| **关系线展开** | 引入配角 | 恩师/亲人/同事，用对话展现立场差异 |
| **压力升级** | 时间更紧/赌注更大 | 又一封律师函/账上只剩3天周转 |

#### 第3章写作重点
```
第3章 = 第二次受阻 + 道德困境 + 关系撕裂 + 引向转折
```

| 模块 | 写作要点 | 具体方法 |
|------|---------|---------|
| **第二次受阻** | 方案行不通 | 用现实规则解释：法律/流程/人情 |
| **道德困境** | 规则vs情感 | 按流程要举报→但举报会毁掉对方<br>合法的方案→但会让某人陷入绝境 |
| **关系撕裂** | 配角与主角冲突 | 一次争吵/冷战/立场对立 |
| **引向转折** | 新信息出现 | 一张旧照片/匿名电话/遗忘的文件 |

---

### 【转】第4章：关键抉择（2000-3000字）

#### 写作重点
```
转 = 证据翻盘 + 压力峰值 + 关键决定 + 行动具体化
```

| 模块 | 写作要点 | 具体方法 |
|------|---------|---------|
| **证据翻盘** | 用客观证据 | 监控录像/账目异常/第三方证言<br>回收之前埋的伏笔<br>**注：必须是关键节点的合理反转，不是悬空设定** |
| **压力峰值** | 全篇最大压力 | 董事会现场/法庭质证/家庭会议<br>倒计时/最后通牒 |
| **关键决定** | 做出选择 | 符合人物性格和前文铺垫<br>不是突然的人设崩塌 |
| **行动具体化** | 用动作不用宣言 | ✅ "他签下名字"<br>❌ "他决定了..." |

#### 禁忌
- ❌ 用巧合翻盘
- ❌ 突然的人设变化
- ❌ 空洞的内心宣言
- ❌ **悬空设定**：用超现实元素（突然的巨额遗产、天降贵人、不可能的运气）解决困境

---

### 【合】第5章：收束与余味（2000-3000字）

#### 写作重点
```
合 = 后果呈现 + 代价不回避 + 关系新平衡 + 留白照应
```

| 模块 | 写作要点 | 具体方法 |
|------|---------|---------|
| **后果呈现** | 不回避代价 | 保住良心→丢了工作<br>守住规则→失去亲情<br>赢了官司→输了名声 |
| **关系新平衡** | 配角的结局 | 用一次对话或行动收束<br>避免全员和解 |
| **生活还要继续** | 日常细节收尾 | 重新投简历/收拾办公桌/回到公交站<br>体现"现实还在，人变了" |
| **留白照应** | 用第1章物件/场景 | 又一次体检通知/又一张裁员名单/又一个清晨 |

#### 禁忌
- ❌ 假团圆（所有矛盾都解决）
- ❌ 过度解释
- ❌ 说教式总结

## 四、人物塑造方法

### 主角设计

| 维度 | 要求 | 正确示例 | 错误示例 |
|------|------|---------|---------|
| **职业** | 具体+中层 | "某公司的财务主管" | "白领" |
| **年龄/位置** | 尴尬期 | 35岁+/临时工转正/部门撤并 | "年轻人" |
| **性格标签** | 2-3个具体特点 | 较真+流程控+怕吵 | "善良" |
| **专业感** | 对话体现职业 | HR说"绩效考核/劳动仲裁"<br>医生说"影像科/会诊/知情同意书" | 所有人说话一样 |

### 压力可量化
```
✅ "月供12000，卡里8000，还有7天"
❌ "经济压力很大"
```

### 配角设计（2-4个功能性配角）

| 类型 | 作用 | 特点 |
|------|------|------|
| **施压者** | 老板/债主/家属 | 有明确利益诉求/有时间限制/对话直接 |
| **共情者** | 同事/朋友/邻居 | 理解但无能为力/提供情感支持 |
| **对立者** | 立场相反的人 | 不是坏人，只是利益不同/有自己的合理性 |
| **见证者** | 旁观者/第三方 | 提供客观视角/伏笔的载体 |

### 关系网设计

| 要素 | 要求 |
|------|------|
| **有历史** | 认识多久/因何结识/重要事件 |
| **有利益** | 经济/职业/情感连接 |
| **有分歧** | 价值观或利益差异 |
| **有变化** | 起稳定→承紧张→转撕裂→合新平衡 |

## 五、爽点设计（现实题材特定）

### 爽点公式
```
现实爽点 = 规则内的反击 + 专业知识的胜利 + 道德高地的占据
```

### 爽点类型

| 类型 | 方法 | 示例 |
|------|------|------|
| **知识型爽点** | 用专业知识解决问题 | 会计找到账目漏洞/律师找到法律依据/医生诊断出误诊 |
| **规则型爽点** | 用对方的规则反制 | 用劳动法反击违法裁员/用合同条款反击甲方 |
| **真相型爽点** | 证据揭示真相 | 监控录像还原事实/证人证言推翻诬陷 |
| **道德型爽点** | 坚守底线获得认可 | 拒绝潜规则反而赢得尊重/举报违规虽有代价但问心无愧 |

### 设计原则
- ✅ 爽点建立在**生活逻辑**之上
- ✅ 依靠**能力和准备**，不是运气
- ✅ 即使是爽点，也要有**代价**
- ❌ 不用超现实力量（突然的贵人相助、不可能的好运）

## 六、冲突设计

### 冲突公式
```
现实冲突 = 外部压力（可量化）+ 内部困境（道德两难）+ 递进层次（表面→关系→价值观）
```

### 外部冲突（主线）

| 压力类型 | 具体化方法 | 示例 |
|---------|-----------|------|
| **金钱** | 数字+deadline | 房贷月供12000/医疗费8万/赔偿金50万 |
| **规则** | 具体条款 | 劳动法第X条/合同第X款/流程规定 |
| **时间** | 倒计时 | 7天后合同到期/今晚8点前/3个工作日内 |
| **关系** | 具体事件支撑 | 恩师带了3年/父母养老/欠了人情 |

### 内部冲突（辅线）

| 冲突类型 | 设计方法 |
|---------|---------|
| **道德困境** | 规则正确 vs 人情难接受<br>自保 vs 牺牲他人<br>真相 vs 善意谎言 |
| **价值观冲突** | 不是"对vs错"<br>而是"对vs对"的选择 |

### 冲突递进层次
```
第1层：表面问题（钱/时间/规则）
    ↓
第2层：关系撕裂（利益冲突/立场对立）
    ↓
第3层：价值观拷问（我是谁/我要什么）
```

## 七、伏笔管理

### 伏笔载体（现实题材特定）

| 载体类型 | 示例 |
|---------|------|
| **纸质物** | 合同/账单/病历/工卡/小票/收据 |
| **电子记录** | 邮件/短信/监控/通话记录/打卡记录 |
| **第三方** | 旁观者/记录者/无意的见证者 |

### 伏笔设置原则

| 阶段 | 方法 |
|------|------|
| **埋设** | 轻触即过，1-2句带过<br>例："他把工卡塞进钱包，和那张褪色的照片挤在一起" |
| **回收** | 证据链还原（多个证据互相印证）<br>证人证言（旁观者提供关键信息）<br>主角顿悟（回想起之前的细节） |
