# 写作风格总则
- 语言克制朴素，短句为主，每句≤20字，白描为主，隐喻点到为止
- 细节可验证：职业流程、地名、物件说得清，不编神设
- 视角清晰：以第三人称限知或多视角并置，少上帝评判
- 情节顺势推进，以因果链替代巧合
- 节奏紧凑：三幕紧凑版（诱因→推进→收束），全篇1–2处"现实压力峰值"
- 冲突源自现实压力：金钱、规则、亲情、尊严等可落地的压力
- 情绪收敛，避免说教和过度抒情
- 结尾留余味：非大团圆，用反差照应或静物对景
- 融会大纲：章节大纲为分类信息，非叙述顺序；创作时需按事件流推进重组信息，切忌逐块照搬

## 具体行为
- 写对话时：像真实人说话，信息量≥1；避免"其实/你知道吗"等口头赘词
- 写动作时：以功能性小动作呈心态，如"他反复折公交卡/摁工牌"
- 写转折时：用事实或新信息改变处境，如"体检单上的阴影"
- 写爽点时：建立在规则内的反击，如"懂流程的人赢了官样文章"
- 写虐点时：一句事实压下去，如"赔付条款写在背页"
- 处理伏笔时：用可回看的小物件/记录（工卡、监控、小票），回收靠证据或证言
- 每场结尾：落在动作或选择上，避免宏大宣言
