
# 用 Claude Agent Skill 写现实题材类短篇小说 - 让故事灵感快速落地

这套方案主要解决一个问题：有了故事灵感后，从构思到完成一个完整短篇小说往往需要几天甚至更久。传统流程是想大纲、设计人物、规划章节、写正文，每一步都需要反复打磨，效率很低。

所以我们通过 Claude 的 Skill 功能搭建了一个专业的创作助手。把现实题材的创作方法论、文档模板、写作风格、实战示例都打包进 Claude Skill，让 Claude 在每个创作环节都能调用这些专业知识。你只需要回答几个问题，剩下的大纲、人物、章节就由 AI 按照专业标准自动生成。

工作流程按照小说创作的自然顺序推进：故事大纲 → 人物小传 → 章节目录 → 章节正文。CLAUDE.md 负责整体流程管控，skill 提供创作标准。从有灵感到生成完整初稿，整个过程可以在一两小时内完成。

这个方案让故事灵感从脑海里的想法变成可以直接阅读的小说初稿，大幅提升创作效率。不过要注意，生成的内容是初稿，还需要根据个人风格润色调整。

# 核心功能模块

**Skill调用机制**

通过 skill 技能包为 Claude 提供专业创作知识。每个创作阶段开始前，主Agent会自动调用对应的资源文件：大纲阶段读取 outline-method.md（方法论）+ outline-template.md（模板）+ outline-example.md（示例），章节阶段读取 output-style.md（写作风格）+ chapter-template.md（模板）。这种模块化设计让 Claude 不是凭感觉写作，而是基于明确的创作标准和范式进行输出。



**分阶段创作流程**

创作严格遵循"故事大纲 → 人物小传 → 章节目录 → 章节正文"的流程，每个阶段对应独立文档。用户通过 /character、/catalog、/write 命令推进流程，系统会自动读取前序文档确保连贯性。比如创作章节时，会同时读取 outline.md、character.md、chapter\_index.md，确保情节不偏离大纲、人物行为符合设定。这种阶段性管理避免了一次性生成导致的前后矛盾。



**模板驱动生成**

所有文档格式由 templates/ 目录下的模板文件定义，包括大纲、人物小传、章节目录、章节正文四类。Claude 创作时严格遵循模板结构，确保输出格式统一、信息完整。比如人物小传必须包含"主要角色→重要配角→其他角色"的层级。



**上下文连贯性**

每个阶段创作前都会读取已有文档内容。人物创作基于大纲设定，章节创作基于大纲+人物+目录，形成递进式上下文积累。系统通过文档驱动而非对话记忆来维护上下文，即使对话被清空，只要文件还在，就能通过 /status 命令查看进度并继续创作。



**文档结构管理**

项目采用标准化目录结构：outline.md（大纲）、character.md（人物）、chapter\_index.md（目录）、chapters/Chapter-XX.md（各章节）。固定5章结构对应起承转合布局，每章2000-3000字。这种结构化存储让创作过程可追溯、可修改，文档本身就是交付物，而不需要从对话记录中复制粘贴整理。



> **注**：AI生成的是初稿，能帮你快速把故事框架搭起来，但最终还需要人工润色才能达到发布标准。输入质量直接决定输出质量——你在Q1-Q3提供的核心创意、冲突设定越详细具体，生成的大纲和人物就越贴合预期。AI的工作是基于你的创意进行填充和扩展，而不是100%从零创造，所以前期构思越清晰，后期调整成本越低。另外，固定的5章结构适合短篇，如果需要更长篇幅，需要调整模板和方法论。

# 如何配置



### 在 Claude Code 中配置

请按照以下步骤配置创作系统，配置完成后，你的项目结构应该是这样：

```python
project/
├── outline.md                           # 故事大纲（创作后生成）
├── character.md                         # 人物小传（创作后生成）
├── chapter_index.md                     # 章节目录（创作后生成）
├── chapters/                            # 章节正文目录（创作后生成）
│   ├── Chapter-01.md
│   ├── Chapter-02.md
│   └── ...
└── .claude/
    ├── CLAUDE.md                        # 项目规则和主Agent配置（必须创建）
    └── skills/
        └── xianshi-skill/               # 现实题材创作技能包（必须创建）
            ├── SKILL.md                 # 技能包核心配置
            ├── outline-method.md        # 大纲创作方法论
            ├── output-style.md          # 写作风格
            ├── templates/               # 文档格式模板
            │   ├── outline-template.md
            │   ├── character-template.md
            │   ├── chapter-index-template.md
            │   └── chapter-template.md
            └── examples/                # 创作示例
                ├── outline-example.md
                ├── character-example.md
                └── chapter-example.md
```



1. **创建项目规则**

* 在 `.claude/` 目录下创建 `CLAUDE.md` 文件

* 粘贴完整的项目规则内容，包含角色定义、工作流程、指令集

* 确保包含初始化脚本和所有创作阶段的流程定义



* **创建 Skill 技能包**

- 在 `.claude/skills/` 目录下创建 `xianshi-skill/` 文件夹

- 创建 `SKILL.md` 作为技能包核心配置

- 粘贴技能包说明和执行流程



* **添加创作方法论**

- 在 `xianshi-skill/` 目录下创建 `outline-method.md`

- 粘贴大纲创作方法论，包含起承转合结构、人物塑造、爽点设计等内容



* **配置写作风格**

- 创建 `output-style.md` 文件

- 粘贴写作风格定义和具体行为规范



* **添加文档模板**

- 在 `xianshi-skill/templates/` 目录下创建四个模板文件

- 包括大纲、人物小传、章节目录、章节正文的格式定义

- 这些模板会严格约束 AI 的输出格式



* **添加创作示例**

- 在 `xianshi-skill/examples/` 目录下创建示例文件



## 重要配置细节：

* **Skill调用**：CLAUDE.md 中定义了何时调用 xianshi-skill，每个创作阶段会自动调用对应资源

* **模板约束**：templates/ 中的格式是强制性的，AI 必须按照模板结构输出

* **方法论指导**：outline-method.md 是核心，定义了起承转合、人物塑造、冲突设计等创作标准

* **风格一致性**：output-style.md 确保文白相济风格贯穿全文，避免出现现代口语

* **示例学习**：examples/ 提供具体范式，让 AI 理解什么是符合标准的输出

* **固定结构**：短篇固定5章对应起承转合，不支持自定义章节数

* **人工润色**：AI生成初稿后仍需人工调整细节、润色语言、补充情节

# 核心设计理念

```plain&#x20;text
┌─────────────────────────┐
│  skill                  │ ← 知识层：提供专业创作指导
│  ├─ outline-method.md   │    方法论、风格、模板、示例
│  ├─ output-style.md     │
│  ├─ templates/          │
│  └─ examples/           │
├─────────────────────────┤
│  CLAUDE.md              │ ← 流程层：控制创作流程
|                         │    定义工作流和指令集
├─────────────────────────┤
│  Main Agent             │ ← 创作层：调用Skill进行创作
|                         │    读取资源→生成内容→写入文档
├─────────────────────────┤
│  outline.md             │ ← 文档层：结构化存储
│  character.md           │    故事大纲
│  chapter_index.md       │    人物小传
│  chapters/*.md          │    章节目录
|                         │    章节正文
└─────────────────────────┘
```



## 工作流程：

1. **需求收集** - 主Agent询问核心创意、核心冲突、故事调性三个问题

2. **调用Skill** - 根据创作阶段，调用skill读取对应的方法论、模板、示例

3. **内容创作** - 基于Skill提供的指导和用户需求，生成符合标准的内容

4. **文档写入** - 将创作内容写入对应的结构化文档（outline.md / character.md等）

5. **流程推进** - 通知用户当前阶段完成，引导进入下一个创作阶段



## 关键特性：

* **Skill驱动** - 每个阶段自动调用对应资源，确保创作有章可循

* **模板约束** - 严格遵循templates/定义的格式，保证输出规范统一

* **阶段递进** - 按照"大纲→人物→目录→正文"顺序推进，后续阶段基于前序文档

* **上下文积累** - 通过文档驱动而非对话记忆维护上下文，支持长期创作

* **结构化存储** - 所有内容分文件管理，便于查看、修改和复用

* **快速落地** - 从灵感到初稿，整个流程可在1-2小时内完成

* **固定框架** - 5章结构对应起承转合，适合快速产出标准化短篇

## **使用时的注意事项：**

1. **输入质量决定输出质量**。Q1-Q3回答得越详细，生成的内容就越贴合你的预期。比如核心冲突如果只写"职场困境"，生成的内容会很空泛；但如果写"38岁HR在裁员名单上看到恩师名字，签字保住职位但违背良心，不签则面临失业和房贷断供"，生成的内容就会具体很多。

2. **这是初稿，不是成品**。AI生成的内容能帮你快速搭起框架，但语言细节、情节逻辑、人物对话还需要人工润色。特别是情感表达部分，AI有时会写得过于直白或抒情，需要调整成更克制、更贴近日常的自然表达。

3. **及时反馈和调整**。如果大纲阶段就发现内容偏离预期，直接提出修改意见，不要等到写完5章再推倒重来。每个阶段都可以修改，改得越早成本越低。

4. **注意版权问题**。虽然这套系统是基于方法论和模板创作，但如果你的故事创意本身借鉴了某部作品，还是要注意避免抄袭。建议用自己的原创构思，或者对经典题材进行充分的改编创新。

> ### 转:选择的时刻(第4章)
>
> 周五下午5点,MCN公司的人坐在菜市场的奶茶店里,合同摊开,等陈默签字。小鱼也在,她给陈默算账:"叔,一年至少50万,比你原来产品经理还赚得多,你还犹豫什么?"
>
> 陈默看着合同条款——每天直播不少于3小时,每周不少于5条视频,内容主题需经公司审核,形象权归公司所有,违约金30万。他想象自己未来三年的样子:每天在镜头前表演"励志的失业者",说着精心设计的台词,卖着批量生产的速食卤味,用"陈默老汤"的标签割一波又一波观众的韭菜。
>
> 他想起三个月前第一次上岗,张姨买完鸭脖,站在摊前说:"小伙子,你这卤味真好吃,有我们那个年代国营食堂的味道。"那是他第一次觉得,这个摊位不只是谋生工具,是一种连接。
>
> 林薇打来电话,陈默说:"我想清楚了。"林薇以为他要签,说:"那就签吧,咱们也该轻松点了。"陈默说:"我不签了,我要关掉直播。"
>
> 电话那头沉默了五秒,林薇说:"你疯了?"陈默说:"我没疯,我想明白了。这三个月我一直在追流量,但流量要的不是我,是那个'励志模板'。我如果签了约,就再也回不去了。"
>
> 林薇问:"那房贷怎么办?女儿学钢琴的钱呢?"陈默说:"我慢慢赚,一天300,一个月也有9000,够了。我把老汤养回来,把老顾客找回来,踏踏实实做这门手艺,总能活下去。"
>
> 林薇哭了,说:"你这个傻子。"但她没再劝。
>
> 陈默走回摊位,小鱼追出来:"叔,你真不签?你知道这个机会有多难得吗?"陈默说:"我知道,但我更知道,我不能为了钱把自己卖了。谢谢你小鱼,是你让我知道我还有价值,但这个价值不该用来演戏。"
>
> 晚上9点,陈默在直播间发了最后一条视频:"各位,谢谢你们这段时间的关注。我决定关掉直播,回归我的小摊位,慢慢做我的老汤。我不适合当网红,我只想做个手艺人。再见。"
>
> 弹幕炸了锅,有人骂他"装清高",有人说"不懂营销",也有人说"尊重"。陈默关掉手机,拆下支架,看着卤锅里的老汤,决定明天开始,重新熬一锅。
>
> ### 合:回到慢生意(第5章)
>
> 一个月后,菜市场32号摊位恢复了平静。陈默每天早上5点起床,文火慢炖72小时的老汤,不赶产量,不做直播,只做本分生意。他重新采购新鲜食材,恢复岳父传下来的配方,一天只卤20斤,卖完就收摊。
>
> 张姨回来了,第一次尝到恢复后的卤味,她说:"这才是那个味道。"李叔也回来了,他还是坐在摊前啃完卤藕再走。慢慢地,老顾客多了起来,陈默认识了更多张脸——带孩子的年轻妈妈、加班回来的外卖员、独居的退休老人——他们不是冲着"网红"来的,是冲着这锅老汤来的。
>
> 收入确实少了,一天300到500,一个月最多15000,比流量时代差远了。林薇没再说什么,她削减了一些开支,女儿的钢琴课停了,改学社区的免费课程。房贷压力还在,但一家人踏实了。
>
> 陈默开始在摊位边放了一张小桌,几个老顾客会坐下来,就着卤味喝点小酒,聊聊天。张姨说这像她年轻时候的国营食堂,李叔说这是"熟人食堂"。陈默喜欢这个称呼,他在摊位上挂了块手写的牌子:"三十二号·熟人食堂"。
>
> 小鱼又来了一次,看着没有直播支架的摊位,说:"叔,你真回去了。"陈默笑:"我本来就该在这儿。"小鱼举起手机:"我能拍一条吗?就拍你做卤味,不要你说话,不要你表演,就拍你的手艺。"陈默想了想,点头:"行,但你得等我这锅卤好,还有两个小时。"
>
> 小鱼坐下来等,陈默给她装了一碟卤豆干。两个小时后,卤味出锅,陈默熟练地分装、切配、摆盘,动作沉稳,不慌不忙。小鱼拍完,说:"叔,这才是你该有的样子。"
>
> 视频发出去,播放量不高,只有8000多,但评论区清一色好评:"这才是真正的手艺人。""看着治愈。""想去买。"陈默看到了,没回复,继续洗他的卤锅。
>
> 冬天来了,菜市场的人流更少了。陈默在摊位前支起炉子,老汤咕嘟咕嘟冒着热气,香味飘出去老远。张姨路过,买了一斤鸭脖,说给孙子带回去。李叔还是坐在摊前,边啃卤藕边和陈默聊天。一个外卖员停下车,要了半斤豆干,说:"师傅,你这卤味是我吃过最踏实的。"
>
> 陈默笑了,他看着那锅老汤,想起岳父说的话:"做吃的,急不得,骗不了人。"
>
> 是的,急不得,骗不了人,也骗不了自己。
>
> 他终于找到了自己的节奏——不是流量的节奏,不是资本的节奏,是这锅老汤的节奏,是市井烟火的节奏,是一个中年人重新站稳脚跟的节奏。
>
> 摊位还在,老汤还在,熟人还在。
>
> 这就够了。

>

## 第1章：三十二号摊位

> 杭州北郊菜市场,早上六点。
>
> 陈默在32号摊位前支起卤锅,点火,调水温。动作熟了,不用多想。这是他失业后的第94天,也是他在这个摊位的第94天。
>
> 卤锅是岳父留下的,铜制,两个耳朵,锅底被火烧得发黑。老汤也是岳父留的,三十年的底子,陈默接手后又养了三个月。文火慢炖,72小时不断,岳父说过,老汤要养,一天都不能断,断了就不是那个味了。
>
> 陈默戴着金属框眼镜,穿白围裙,手指有卤料的痕迹,洗不掉。他今年42岁,三个月前还是某互联网公司产品经理,年薪38万,现在一天赚300到500。
>
> 锅里的汤汁开始冒泡,香料的味道散出来。陈默往锅里下了鸭脖,十斤,够卖一上午的。他看了眼手机,早上6点12分,林薇发来消息:"这个月房贷还差8000,女儿钢琴课续费也快到了。"
>
> 陈默没回。他算过账,房贷月供15000,林薇工资6000,他现在一天净赚300出头,一个月9000,缺口6000。加上女儿钢琴课3800,生活开销,缺得更多。
>
> 他把手机翻过来,屏幕朝下扣在案板上。
>
> 早上7点,第一批顾客来了。
>
> 张姨,58岁,退休纺织厂工人,头发花白,手上有老茧。她每天早上买半斤鸭脖,带走时陈默会多装两块豆干。
>
> "小陈,今天的好了没?"张姨拎着保温袋,站在摊位前。
>
> "还得再卤二十分钟。"陈默说,"您先逛逛,一会儿再来。"
>
> "那我去买菜,回头来拿。"张姨走了,又回头,"对了,你记得我孙子不吃姜。"
>
> "记得。"陈默说。
>
> 他记得。张姨孙子7岁,不吃姜,也不吃花椒。张姨每次买鸭脖都会提醒一遍,陈默每次都会挑干净了再装袋。三个月下来,他记住了十几张脸,也记住了他们的口味习惯。
>
> 8点,李叔来了。63岁,退休泥瓦匠,黝黑精瘦,穿旧工装,走路有点瘸。他爱吃卤藕,每次都在摊前坐着啃完再走。
>
> "老李,卤藕还没好。"陈默说,"得中午。"
>
> "我知道。"李叔在摊位边的小凳子上坐下,点了根烟,"先买半斤豆干。"
>

> 弹幕更多了:"不要走!""我们支持你!""为什么?"
>
> "我要回归我的小摊位,慢慢做我的老汤。"陈默说,"72小时慢炖,一天都不能断。这是我岳父留下的手艺,我不能为了流量把它毁了。"
>
> 弹幕开始分化。
>
> 有人骂:"装清高。""不懂营销。""浪费流量。"
>
> 也有人说:"尊重。""理解。""做你自己。"
>
> 陈默看着这些弹幕,笑了笑。
>
> "再见,各位。"他说。
>
> 关播。
>
> 在线人数从3.2万瞬间归零。
>
> 陈默放下手机,拆下支架。
>
> 摊位前又安静了。没有补光灯,没有麦克风,没有摄像头。
>
> 只有他,和那锅老汤。
>
> 陈默走到卤锅前,看着锅里的汤汁。
>
> 水面浑浊,香料漂着,已经看不出老汤的底色了。
>
> 岳父留下的汤底,被他稀释了三次,现在只剩一点点了。
>
> 陈默关了火,端起锅,把汤倒进水槽。
>
> 老汤流进下水道,带着香料,带着那点仅存的底子,全部倒光了。
>
> 他把锅刷干净,铜锅在水里泛着光。
>
> 这是岳父用了三十年的锅,锅底被火烧得发黑,但还是结实。
>
> 陈默决定,从明天开始,重新熬一锅老汤。
>
> 72小时,文火慢炖,一天都不能断。
>
> 他要把老汤养回来,把老顾客找回来,把自己找回来。
>
> 陈默锁上摊位,往回走。
>
> 杭州的夜晚很安静,菜市场已经空了。路灯昏黄,照在空荡荡的街道上。
>
> 陈默掏出手机,看了看银行卡余额。
>
> 4.2万。
>
> 这是他直播一个半月赚的,比摆摊三个月还多。
>
> 但从明天开始,他又要回到一天300的日子了。
>
> 房贷15000,林薇工资6000,他9000,还差几千。
>
> 女儿钢琴课停了,开销要省着点了。
>
> 但陈默觉得踏实。
>
> 他想起岳父说的话:"做吃的,急不得,骗不了人。"
>
> 是的,急不得,骗不了人,也骗不了自己。
>
> 陈默把手机关机,抬头看了看天。
>
> 杭州的夜空,星星很少,但月亮很亮。
>
> 他深吸了口气,往家的方向走去。
>
> 明天,他要重新开始。

## 第5章：熟人食堂

> 一个月后。
>
> 杭州北郊菜市场,早上6点。陈默在32号摊位前支起卤锅,点火,调水温。
>
> 动作又熟悉起来了,不急不慢。
>
> 锅里是新熬的老汤,文火慢炖了72小时,一天都没断过。汤色浓郁,香味醇厚,是岳父留下的那个味道。
>
> 陈默往锅里下了鸭脖,20斤,够卖一上午。他不赶产量了,一天就做20斤,卖完收摊。
>
> 摊位恢复了原来的样子。没有手机支架,没有补光灯,没有麦克风。只有卤锅,案板,小凳子,还有一张新加的小木桌。
>
> 早上7点半,张姨来了。
>
> "小陈,今天的好了没?"张姨拎着保温袋,站在摊位前。
>
> "再十分钟。"陈默说。
>
> "那我先去买菜,回头来拿。"张姨说。
>
> 十分钟后,张姨回来了。陈默给她装了半斤鸭脖,挑干净姜,多放了两块豆干。
>
> 张姨接过袋子,拿起一块鸭脖闻了闻,咬了一口。
>
> "这才是那个味道。"她说。
>
> 陈默笑了。
>
> "我这个月每天都来别的摊买,吃了一圈,还是你这个好吃。"张姨说,"我就说嘛,老汤不能急。"
>
> "是,不能急。"陈默说。
>
> 张姨走了,背影还是那样,但脚步轻快了些。
>
> 8点,李叔来了。他在摊前坐下,要了半斤卤藕。
>
> 陈默给他装好,李叔拿起一块啃了啃,点了点头。
>
> "对了,这个味。"李叔说。
>
> 他还是坐在摊前啃完,啃完用袖子擦嘴,点根烟。
>
> "小陈,你那个直播不做了?"李叔问。
>
> "不做了。"陈默说,"做不来。"
>
> "做不来就对了。"李叔说,"做吃的和唱戏不一样,不能给人看热闹。"
>
> 陈默笑了。他现在明白这个道理了。
>
> 慢慢地,老顾客又多起来了。
>
> 带孩子的年轻妈妈,每次买一斤鸭脖,说给孩子当零食。加班回来的外卖员,要半斤豆干,说垫垫肚子。独居的退休老人,买几块卤藕,坐在摊前慢慢吃。
>
> 他们不是冲着"网红"来的,是冲着这锅老汤来的。
>
> 陈默认识了更多张脸,也记住了更多习惯。谁不吃辣,谁爱吃软的,谁要切小块,他都记得。
>
> 收入确实少了。
>
> 一天300到500,一个月最多15000,比直播时代少了一半多。
>
> 林薇削减了开支。女儿的钢琴课停了,改上社区的免费音乐课。周末不出去吃饭了,在家做。衣服也不怎么买了,穿旧的。
>
> 但没人抱怨。
>
> 女儿在社区音乐课上学了竖笛,每天在家吹,虽然跑调,但开心。林薇周末会来摊位帮忙,和张姨李叔聊天,说这里像她小时候的菜市场。
>
> 房贷还是压力,每个月都得算账,有时候差几百,陈默就多做一天。
>
> 但一家人踏实了。
>
> 十月中旬,陈默在摊位边加了张小木桌,放了几个小凳子。
>
> 傍晚,有老顾客会坐下来,就着卤味喝点小酒,聊聊天。
>
> 李叔说自己年轻时在工地干活,一天搬200块砖。张姨说她在纺织厂上班,机器声震得耳朵疼。外卖员小王说他一天送80单,最远跑到西湖边。
>
> 陈默也会坐下来,听他们说话,偶尔插两句。
>
> 有时候不说话,就坐着,看着卤锅里的汤汁慢慢翻滚。
>
> 张姨说,这像她年轻时候的国营食堂,下班了大家坐在一起吃饭,聊天,不急着走。
>
> 李叔说,这是"熟人食堂"。
>
> 陈默喜欢这个称呼。
>
> 他找了块木板,用毛笔写了几个字:"三十二号·熟人食堂"。字写得不好,歪歪扭扭的,但他还是挂在了摊位上。
>
> 林薇来看了,说:"这字丑的。"
>
> "能看清就行。"陈默说。
>
> 十一月初,小鱼又来了。
>
> 她站在摊位前,看着没有直播支架的摊位,说:"叔,你真回去了。"
>
> "我本来就该在这儿。"陈默说。
>
> 小鱼点点头,举起手机:"我能拍一条吗?"
>
> 陈默愣了一下。
>
> "就拍你做卤味,不要你说话,不要你表演,就拍你的手艺。"小鱼说。
>
> 陈默想了想,点头:"行,但你得等我这锅卤好,还有两个小时。"
>
> "我等。"小鱼说。
>
> 她在小木桌旁坐下,陈默给她装了一碟卤豆干。
>
> 小鱼吃着豆干,看着陈默在卤锅前忙活。调火候,下料,翻动,盖锅盖。动作很慢,但每一步都稳。
>
> 两个小时后,卤味出锅。
>
> 陈默熟练地分装,切配,摆盘。动作沉稳,不慌不忙,没有多余的话。
>
> 小鱼举着手机拍,没说话,也没加特效。
>
> 拍完,她说:"叔,这才是你该有的样子。"
>
> 陈默笑了:"我一直都是这个样子。"
>
> 小鱼把视频发了,没加标题,只写了三个字:"32号摊"。
>
> 播放量不高,只有8000多,但评论区清一色好评。
>
> "这才是真正的手艺人。"
>
> "看着治愈。"
>
> "想去买。"
>
> 陈默看到了,没回复,继续洗他的卤锅。
>
> 冬天来了。
>
> 杭州的冬天不算冷,但早晚温差大。陈默在摊位前支起炉子,老汤咕嘟咕嘟冒着热气,香味飘出去老远。
>
> 菜市场的人流少了,但32号摊位还是有人来。
>
> 张姨还是每天早上来,买半斤鸭脖,给孙子带回去。
>
> 李叔还是坐在摊前,边啃卤藕边和陈默聊天。
>
> 外卖员小王送完单会绕过来,要半斤豆干,说:"师傅,你这卤味是我吃过最踏实的。"
>
> 有时候会有新顾客,看到"三十二号·熟人食堂"的牌子,好奇地走过来。
>
> "老板,这是食堂吗?"
>
> "不是,是卖卤味的。"陈默说,"但你可以坐下来吃。"
>
> 新顾客买了卤味,坐在小木桌旁吃。吃完说好吃,问下次还能来吗。
>
> "随时来。"陈默说。
>
> 有的新顾客来了几次,就变成了老顾客。陈默又记住了几张新面孔,也记住了他们的习惯。
>
> 十二月中旬,林薇带着女儿来摊位。
>
> 女儿拿着竖笛,说要给大家吹一首。
>
> "吹吧。"陈默说。
>
> 女儿吹了首《小星星》,音跑了,节奏也不对,但大家都在鼓掌。
>
> 张姨说:"吹得好!"
>
> 李叔说:"有出息!"
>
> 女儿开心地笑了,说:"爸爸,我以后也要像你一样,做个手艺人。"
>
> 陈默摸了摸她的头:"好,但你得先把竖笛吹好。"
>
> 傍晚,摊位收了。
>
> 陈默清洗卤锅,林薇在旁边收拾案板。女儿坐在小凳子上,吹着跑调的竖笛。
>
> "今天赚了多少?"林薇问。
>
> "380。"陈默说。
>
> "够吗?"
>
> "够。"陈默说,"慢慢来,够的。"
>
> 林薇没说话,继续收拾。
>
> 陈默把卤锅洗干净,看着铜锅在水里泛着光。
>
> 锅底还是被火烧得发黑,但锅里的老汤又养回来了。72小时慢炖,一天都没断过,味道越来越醇。
>
> 他想起岳父说的话:"做吃的,急不得,骗不了人。"
>
> 是的,急不得,骗不了人,也骗不了自己。
>
> 陈默锁上摊位,和林薇女儿一起往回走。
>
> 杭州的冬夜很安静,菜市场早就关门了。路灯昏黄,照在空荡荡的街道上。
>
> 陈默回头看了一眼摊位。
>
> "三十二号·熟人食堂"的牌子还挂着,在灯光下晃了晃。
>
> 他笑了。
>
> 他终于找到了自己的节奏——不是流量的节奏,不是资本的节奏,是这锅老汤的节奏,是市井烟火的节奏,是一个中年人重新站稳脚跟的节奏。
>
> 摊位还在。
>
> 老汤还在。
>
> 熟人还在。
>
> 这就够了。
